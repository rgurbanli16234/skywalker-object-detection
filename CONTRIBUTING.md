# Contributing to Skywalker Object Detection

We welcome contributions to make the Skywalker Object Detection pipeline more robust.

## Development Setup
1. Fork the repository.
2. Clone your fork.
3. Install dependencies: `pip install -r requirements.txt`.
4. Create a feature branch: `git checkout -b feature/your-feature`.

## Pull Request Process
1. Ensure your code passes all linting (`flake8` / `black`).
2. Update the `CHANGELOG.md` with a note of your changes.
3. Request review from maintainers.
